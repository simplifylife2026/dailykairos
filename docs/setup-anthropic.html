Setup · AI Provider

Anthropic (Claude)

Connect Daily Kairos to Anthropic to use Claude — Anthropic's flagship AI model — for generating thoughtful, nuanced Bible reflections. You'll need an Anthropic Console account and an API key.

🌐 Step 1 — Get your Anthropic API key

Create or sign in to your Anthropic account at console.anthropic.com, then navigate to API Keys to generate a new key. Copy it — Anthropic only shows it once.

Anthropic API key documentation ↗
⚠️ New accounts require credits
New Anthropic accounts must add billing credits before the API is active. Go to Settings → Billing in the Console and add a small amount (e.g. $5) to get started.

📱 Step 2 — Enter the key in Daily Kairos

2

Open Daily Kairos, tap the Settings icon (⚙️), then tap AI Provider. Tap Add Provider.

3

Select Anthropic Claude from the provider menu.

4

Paste your API key into the API Key field.

5

Tap the Model field and choose a model from the dropdown (e.g. claude-3-5-haiku for speed, or claude-opus-4 for deeper reflections).

6

Tap Verify & Save. If the key is valid and your account has credits, Anthropic Claude will be saved and set as your active provider. If there's an error, check your key and billing status.

💳 Billing & usage
Anthropic charges per token. A typical Daily Kairos reflection costs a small fraction of a cent. Monitor usage and set limits at console.anthropic.com → Settings → Billing.
